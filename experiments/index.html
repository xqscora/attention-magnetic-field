MFA Experiment Suite

Three empirical tests of the Magnetic Field Model of Attention (Zeng, 2026)
Pre-registered on OSF · Preprint on Zenodo

1

Attentional Gradient Decay Function

Tests whether spatial attention follows a power-law decay F = S/rα rather than Gaussian or exponential. Modified Posner cueing paradigm with targets at 8 eccentricities.

640 trials ~35 min Prediction 1
2

Continuous Load–Distractor Function

Tests whether distractor interference decreases as a continuous sigmoid (not binary step) function of perceptual load. Modified Lavie flanker task with 8 parametric load levels.

960 trials ~45 min Prediction 2
3

Overexcitability × Distractor Interaction

Tests whether individual OE profiles predict dimension-specific distractor vulnerability. OEQII questionnaire followed by letter identification with five distractor types (sensory, semantic, emotional, movement, neutral).

OEQII + 300 trials ~45 min Prediction 3
